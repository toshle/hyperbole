# hyperbole
Hyperbole sentiment analysis with python

# What is needed
* `pip install nltk`
  1. run `python`
  2. `>>import nltk`
  3. `>>nltk.download()`
  4. Go to All Packages and download `punkt`
* `pip install tweepy`
* `pip install flask`

# Start
* `python app.py`
* Then http://127.0.0.1:5000/<number of tweets to retrieve>
